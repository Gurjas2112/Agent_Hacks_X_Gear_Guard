GearGuard - The Ultimate Maintenance Tracker ============================================ A comprehensive maintenance management system for tracking company assets (machines, vehicles, computers) and managing maintenance requests. Core Features ------------- **Equipment Management** - Central database for all company assets - Track by Department or Employee - Dedicated Maintenance Team assignment - Purchase & Warranty tracking - Location management **Maintenance Teams** - Define specialized teams (Mechanics, Electricians, IT Support) - Assign team members/technicians - Team-based workflow restrictions **Maintenance Requests** - Corrective (Breakdown) requests - Preventive (Routine) maintenance - Auto-fill from equipment selection - Stage-based workflow (New → In Progress → Repaired → Scrap) - Duration tracking Key Views --------- - **Kanban Board**: Drag-and-drop with overdue indicators - **Calendar View**: Schedule preventive maintenance - **Pivot/Graph Reports**: Analyze requests by team/category Smart Features -------------- - Smart buttons with request counts - Automatic team/category population - Scrap logic with equipment flagging - Overdue request highlighting Author: GearGuard Team License: LGPL-3
